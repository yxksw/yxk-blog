## 模块化路由

将每个业务功能的借口拆分到独立文件里，再统一挂载到主应用中

```python
from fastapi import APIRouter


# 创建 APIRouter 实例
# prefix 路由前缀
# tags 分组 标签
router = APIRouter(prefix="/api/news", tags=["news"])

@router.get("/categories")
async def get_categories():
    return {"message": "获取分类成功"}

```
